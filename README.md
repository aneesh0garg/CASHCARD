### CashCard service for Family Cash Card

**Build your project**
`./gradlew build`

**Run unit tests**
`./gradlew test`

For more verbose console output:
`./gradlew test --info --console=plain`
`./gradlew test --debug --console=plain`
`./gradlew test --stacktrace --info --console=plain`

**Show test reports**
`open build/reports/tests/test/index.html`
